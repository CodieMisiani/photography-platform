# 📸 Photography Business Platform

A modern photography business platform built for event photographers.

## Features

- Portfolio showcase
- Booking system
- Availability calendar
- Quote requests
- Admin CMS
- Invoice generation
- M-Pesa Daraja STK Push payments

## Tech Stack

### Frontend

- React (Vite)
- Tailwind CSS
- React Router

### Backend

- Node.js
- Express.js

### Database

- PostgreSQL

### Session Management

- Redis

### Payments

- Safaricom Daraja API

## Development Methodology

This project follows:

- Software Development Life Cycle (SDLC)
- Agile Methodology
- 2-week sprint planning
